# Monitoring Policy

## Metriken

- Request Count
- Error Rate
- Latency
- Token Usage
- Cost per Request
- Tool Success Rate
- Agent Success Rate
- Retry Count
- Timeout Count
- Conversion- und Lead-KPIs

## Betrieb

Produktionsnahe Services benoetigen Health-, Readiness- und Metrics-Konzepte oder eine begruendete Ausnahme.
