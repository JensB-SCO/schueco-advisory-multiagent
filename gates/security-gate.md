# Security Gate

## Prueffragen

- Werden Secrets, personenbezogene Daten oder geschuetzte Systeme beruehrt?
- Sind Authentifizierung, Autorisierung und Rate Limits relevant?
- Gibt es Prompt-Injection-, Tool-Misuse- oder Datenabflussrisiken?
- Sind Audit Logging und Datenschutzanforderungen beschrieben?

## Mindestanforderung

Die Empfehlung muss Security-Risiken, Datenschutzsicht und notwendige Schutzmassnahmen nennen.
